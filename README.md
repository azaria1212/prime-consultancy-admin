# Prime Consultancy PLC - Admin Dashboard

Professional admin dashboard for managing the Prime Consultancy website.

## 🚀 Features

- ✅ Login/Authentication
- ✅ Dashboard with Statistics
- ✅ Appointments Management
- ✅ Contact Messages
- ✅ Blog Management
- ✅ Services Management
- ✅ Case Studies Management
- ✅ Testimonials Management

## 📦 Installation

```bash
cd admin
npm install
```

## 🏃 Run

```bash
npm run dev
```

```bash

npm install -g vercel

# Deploy
```
